Weasley project

Welcome to this new 2-player adventure!

Will you be able to save the world?



To play, you need to go to the starter map with an even number of people. Everyone must enter the red zone, and wait for the administrator to start the draw.

The administrator is someone who goes to the url of this map with "#admin-entry" to launch the draw that will send players to their instance.


Start the adventure


NOTE : To play with your friends, you must go on the same instance of the game. To make people play at the same time, the "Start the adventure" button creates a new instance for all players, so send your link to your friends !

NOTE : This adventure was created to be played on a computer. It is not intended for cell phones. In order to have the best experience possible, please use a computer.
